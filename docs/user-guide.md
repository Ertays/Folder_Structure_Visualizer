## Приступая к работе
### Запуск приложения
1. Дважды щелкните мышью `src.sln`
2. Приложение автоматически запустится на вашем диске C:
### Настройка в первый раз
Настройка не требуется! Приложение готово к использованию немедленно.

### Элементы интерфейса
- **Текущий путь**: Показывает выбранный в данный момент каталог
- **Информация о диске**: Отображает все доступные диски в вашей системе
- **Список файлов**: Показывает все файлы и папки в текущем каталоге
- **Кнопки управления**: Выполняют различные операции с файлами
## Основные операции
### Просмотр файлов и папок
1. Приложение автоматически загружает содержимое при запуске
2. Файлы имеют префикс [ФАЙЛ]
3. Папки имеют префикс [ПАПКА]
4. Внизу отображается общее количество файлов.
### Обновление списка файлов
- Нажмите кнопку **Обновить**, чтобы обновить список файлов
- Полезно после внесения внешних изменений в каталог
### Переключение Дисков
1. Нажмите **Изменить диск**
2. Приложение переключается между диском C: и другими доступными дисками
3. Информация о диске обновляется автоматически
## Файловые операции
### Создание новых файлов
1. Нажмите кнопку **Добавить файл**
2. Автоматически создается новый текстовый файл (например, `MyFile1.txt`).
3. Файл содержит образец текстового содержимого
4. Счетчик файлов увеличивается автоматически
### Создание новых папок
1. Нажмите кнопку **Добавить папку**
2. Автоматически будет создана новая папка (например, `MyFolder1`)
3. Счетчик папок автоматически увеличивается
### Renaming Items
1. Нажмите **Переименовать**
2. Введите текущее имя файла/папки
3. Введите новое имя
4. Нажмите "ОК" для подтверждения
### Удаление элементов
1. Нажмите **Удалить**
2. Введите название файла/папки для удаления
3. Нажмите **ОК** для подтверждения
4. **Внимание**: Удаленные элементы восстановить невозможно!
