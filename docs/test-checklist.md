<h1>Test Checklist - Folder Structure Visualizer</h1>
</div>

## Тестовая среда
- **OS**: Windows 11
- **.NE**T Version: 8.0
- **Architecture**: x64
## ✅ Контрольный список ручных тестов
### 1. Запуск приложения
- Приложение запускается без ошибок
- Главное окно отображается корректно
- В строке заголовка отображается "Визуализатор структуры папок"
- Окно расположено по центру экрана
### 2. Обнаружение Дисков
- Доступные диски указаны правильно
- При запуске текущего пути отображается **C:**
- Информация о диске отображает все готовые диски
- Неготовые диски не отображаются
### 3. Отображение списка файлов
- Отображаются файлы и папки из **C:**
- Папки имеют префикс **[FOLDER]**
- Файлы имеют префикс **[FILE]**
- Общее количество точно указано
- Когда содержимое превышает размер поля, появляется полоса прокрутки
### 4. Обновите функциональность
- Кнопка Обновить обновляет список файлов
- После обновления появляются новые файлы
- Удаленные файлы исчезают после обновления
- Значения счетчиков обновляются корректно
## 5. Файловые операции
### Добавить файл
- Создает новый файл с правильным именем (**MyFile1.txt**)
- Счетчик файлов увеличивается (**MyFile2.txt**, **MyFile3.txt** и т.д.)
- Файл содержит ожидаемое содержимое
- Появляется сообщение об успешном завершении
- Файл появляется в списке после создания
### Добавить папку
- Создает новую папку с правильным именем (**MyFolder1**)
- Счетчик папок увеличивается (**MyFolder2**, **MyFolder3** и т.д.)
- Появляется сообщение об успешном завершении
- Папка отображается в списке после создания
### Переименовать
- Успешно переименовывает файлы
- Успешно переименовывает папки
- Обработка ошибок для несуществующих элементов
- Список обновляется после переименования
### Delete
- Успешно удаляет файлы
- Успешно удаляет папки (включая содержимое)
- Обработка ошибок для несуществующих элементов
- Обновление списка после удаления
- Выводится предупреждение о недопустимых именах
